# Default fallback font sources

These fonts are bundled for the hosted SD-card `.cpfont` builder. They are used
after the user's primary font and any user-uploaded fallback families, only for
selected codepoints that earlier fonts do not contain.

All bundled fonts are distributed under the SIL Open Font License 1.1. See
`OFL.txt` in this directory.
